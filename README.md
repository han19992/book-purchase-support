# 도서 구매비 지원 프로그램

12명 팀원이 함께 쓰는 도서 구매 신청/상태 관리 페이지입니다.

## 바로 배포하는 방법

이 저장소는 Vercel에 그대로 가져가면 되는 정적 사이트입니다.

1. Vercel 대시보드에서 새 프로젝트를 만듭니다.
2. 이 GitHub 저장소를 import 합니다.
3. 별도 빌드 설정 없이 배포합니다.

루트의 `index.html`이 바로 시작 화면입니다.

## 포함 기능

- 프로그램 시작 월, 마감일, 담당자 안내
- 도서 신청 등록
- 구매 상태 관리
- 공유 가능 상태 관리
- 개인 배송지 비밀번호 열람
- CSV 다운로드
- 브라우저 저장 기반 데이터 보관

## 로컬 확인

필요하면 아래처럼 로컬 서버로도 확인할 수 있습니다.

```bash
/Users/han/.cache/codex-runtimes/codex-primary-runtime/dependencies/python/bin/python3 app.py
```

그다음 `http://127.0.0.1:8000`을 엽니다.

## 참고

- 공개 저장소 기준으로 민감한 값은 넣지 않았습니다.
- Google Sheets 연동은 별도 서버 연동이 있어야 하므로, 현재 버전은 브라우저 기준으로 관리됩니다.
